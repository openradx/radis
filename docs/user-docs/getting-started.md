# Getting Started

Welcome to RADIS! This guide will help you get started with using RADIS for radiology report management and search.

## First Login

1. Open your web browser and navigate to the RADIS URL provided by your administrator
2. Enter your username and password
3. Click "Login" to access the RADIS dashboard

## Understanding Groups

RADIS uses groups to organize reports and control access:

- Your user account is assigned to one or more groups
- You can only view reports that belong to your active group
- Change your active group in your user profile settings
