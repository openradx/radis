Extractions Help

Extractions assist you in fetching data from radiology reports into a tabular format for detailed analysis. This process leverages both keyword-based searches and a Large Language Model (LLM) to filter and extract relevant information.

To create a new extraction job, these three steps must be followed:

  1. Search for reports using a keyword-based query and database filters.
  2. Define filters that are used by a LLM to further refine the results.
  3. Specify the data that the LLM should finally extract from the report.
